**Course:** WEB 100–106

This Weather App was built using HTML, CSS, and JavaScript. I created the page structure with HTML, styled it with CSS to match the required design, and used JavaScript to make the app interactive. The app allows a user to search for a city, then it uses the Open-Meteo Geocoding API to get the city’s latitude and longitude. After that, it uses the Open-Meteo Weather API to fetch and display the current weather, humidity, wind speed, UV index, and a 5-day forecast.
